# CRISPR Disease Gene Analysis

This project explores the biological significance of the TP53 tumor suppressor gene and its role in cancer biology. The goal of this project is to understand how genomic databases and bioinformatics tools can be used to study disease-associated genes and explore their relevance in modern genome editing research.

## Objectives

- Investigate the genomic location and structure of the TP53 gene
- Analyze evolutionary conservation of TP53 using BLAST
- Explore mutation data using ClinVar and COSMIC databases
- Understand how genome editing technologies such as CRISPR can be used to study disease-related genes

## Gene of Interest

**TP53** is a tumor suppressor gene that encodes the p53 protein, which plays a key role in maintaining genomic stability by regulating the cell cycle, DNA repair, and apoptosis.

Mutations in TP53 are found in approximately 50% of human cancers, making it one of the most important genes studied in cancer biology.

## Tools and Databases Used

- NCBI Gene
- NCBI BLAST
- ClinVar
- COSMIC
